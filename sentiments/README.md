### Requirements
```
pip3 install --user -r requirements.txt
```

### Usage
```
$ ./smile "I love you"
:)
$ ./smile "I hate cats"
:(
$ ./smile "Stanford"
:|
```

```
export API_KEY=value
export API_SECRET=value
$ ./tweets @cs50
 0 hello, @world
 1 I love you, @world
-1 I hate you, @world
...
```

```
$ flask run
```
